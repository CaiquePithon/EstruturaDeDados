# 📚 Estrutura de Dados

Repositório dedicado ao armazenamento e organização dos códigos desenvolvidos durante as aulas de **Estrutura de Dados**.  
Aqui você encontrará implementações práticas, exercícios e experimentações com diferentes estruturas fundamentais da computação.

---

## 🧠 Sobre o Projeto

Este repositório tem como objetivo consolidar o aprendizado em Estrutura de Dados por meio da prática, abordando conceitos essenciais utilizados no desenvolvimento de software eficiente e escalável.

Os códigos aqui presentes foram desenvolvidos ao longo das aulas, servindo como material de estudo e consulta futura.
